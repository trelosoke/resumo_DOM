Resumindo DOM

Definição

DOM(Document Object Model) é um modelo de objeto usado para documentos (sites). Isso que permite a inserção do JavaScript nos navegadores. Dá acesso aos componentes internos do site (body, div, input, etc).

Àrvore DOM

O elemento base do DOM, onde tudo se ramifica, é window. Dentro de window temos diversos atributos. Dentre eles, o que dá acesso ao site é chamado document. É nesse elemento onde se tem acesso a todo o código HTML.

Acesso aos Elementos

No JavaScript temos diversas formas de acessar os elementos, especificamente cinco formas:

  • Por Marca (Tag Name)

    • getElementsByTagName()[]: seleciona mais de um objeto. É amplo. Caso, por exemplo, o meu tag name fosse uma tag 'p', como provavelmente há mais de um 'p' no meu código eu teria que dizer qual é a que eu quero. Isso é possível com "[ ]" ao final do Tag Name, especificando numericamente qual o elemento, sendo "0" o primeiro que aparece no código e assim sucessivamente.

  • Por ID

    • getElementById(): usar para identificar um único elemento, de forma exclusiva.

  • Por Nome

    • getElementsByName()[]: usar para inputs de formulários que compartilham o mesmo nome.

  • Por Classe

    • getElementsByClassName()[]: para vários elementos com mesmo estilo ou comportamento.

  • Por seletor (Atual)

    • querySelector()/querySelectorAll(): Pode ser usado para selecionar tanto a classe quanto o ID do elemento. Basta colocar "querySelector(elemento#id)" para id ou "querySelector(elemento.classe)" para classe.

Eventos DOM

Evento DOM é tudo aquilo que pode acontecer com um elemento. Clicar, mover,segurar, apertar tecla específica, segurar, entre outras coisas.

Funções

Funções são tipos primitivos que guardam um código (bloco {}) que só rodará se um evento acontecer. Geralmente é usada assim: 


    function {

    }

Ou desse jeito:


    function evento(){
        
    }
